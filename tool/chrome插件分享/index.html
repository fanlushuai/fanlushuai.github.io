Chrome插件分享

chrome提供了丰富的插件库。如果想要发挥chrome潜力，安装各种插件是必不可少的。本文分享一下作者常用的插件。每一个插件带来的体验提升都是令人激动的。

1. 扩展管理器

可以快速的管理扩展，开启和关闭。能够配置情景模式，一键切换扩展组的启用和关闭。另外插件提供了比较实用的工具，可以在特定的网站开启插件。能够节省内存，在不需要的时候，自动开启和关闭插件。比如，github的 Octree。在github网站打开的时候，才会打开插件。

2. infinity pro

新标签页。非常好用。新标签页如同变成小手机。

3. toby mini

这个是一个tab存储的插件，能够在打开很多标签的时候，save session，保存所有当下的标签，方便下次打开。mini版不会占用新标签页。可以和infinity配合使用

4. 书签侧边栏

能够管理书签。在浏览器的侧边栏。这样可以将chrome浏览器的浏览区域扩大。视觉上也很舒服。

5. AdGuard

广告拦截插件。内存占用量是adblock的一半左右。功能上基本一样。所有可以赶快干掉adblock已经adblock plus

6. LastPass

一款密码管理插件。主动询问保存域名下的密码，并且在登录的时候能够自动帮助填充。拥有账号体系，可以非常方便的在不同的账号下使用。

7. 一键切换

使用搜索引擎的时候，使用快捷键的形式，快速的切换另外一种搜索引擎来搜索现在的关键字。

8. Surfingkeys

和vimium类似的强大工具。如果习惯使用vimium的，强烈推荐使用。能够有v模式。和更强大的bookmark，history搜索翻页的能力。还有支持自定义快捷键和一些自定义的脚本。个人自定义了一些东西，可以参考一下

9. 有道词典划词扩展2019

个人认为比较好用的·翻译插件。

10. save-data-header

能够在每个http头上添加accept-coding,告诉服务器，可以返回br格式的压缩报文。可以显著的减少支持br格式压缩的网站的数据报文。加速打开。

11. Switchy omega

代理切换。

12. Octotree

github使用的查看仓库源码文件的目录结构。

13. GitCode Tree

码云提供的类似Ocotree。查看仓库源码文件的目录结构

14. tampermonkey

油猴插件。能够注入js脚本。拥有脚本库。有大量编写好的插件，也可以自己写脚本注入。常用的脚本入各大vip视频网站的破解。

15. quick javascript switcher

有些网站不支持复制。直接禁用js。大多数就立马投降了。

16. Stylus

提供了css库，能够改变某些网站的css布局和效果。我经常使用google center search result。使得搜索结果居中。很舒服。

17. xpath helper

写xpath的时候，可以用他来辅助验证自己的xpath效果。

18. easy auto refresh

自动刷新页面。

19. editthiscookie

这个你看着用。

20. separate window

基本上算是个摸鱼的神器。能够将网页里面的视频扣出来播放。一一个新的小窗。

21. 购物党

显示一些商品的历史价格，在京东，天猫等购物的时候不至于被坑。

22. Bookmarks clean up

书签清理工具。移除重复的，空文件夹，失效的

23. 下载管理

chrome自带的下载管理，设计及其影响全局视觉。通过这个插件可以通过点击插件图标方便查看下载状态。也不会出现原有的下载底栏影响视觉。注意，下关管理的通知使用了windows的通知。系统可能因为专注模式，或者通知关闭导致没有通知。

24. astrolabe

可以用快捷键ctrl+m预览所有tab的缩略图。在打开很多tab的时候，可以方便的找到需要的tab。（可以通过chrome自带的扩展的快捷键设置来修改触发的快捷键。本人改成了ctrl+b.有利于单手操作）

25. darkreader

可以调节网页的亮度，和对比度，找到自己合适的效果。颜色的话，只有黑白，但是亮度和对比度，调节好了之后，基本上颜色也没有啥要求了。因为已经很舒服 了。如果你有绿色背景的需求，这里可以不用担心

26. simpRead 简悦

正文提取算法。提取正文。排除其他信息干扰。阅读更加舒服。

27. Tab Activate

使用新标签页打开所有链接。实际上如果，熟练使用ctrl按键。来控制使用新标签打开新标签。方法更加的灵活适用。

28. Video Speed Controller

加速HTML5 audio/video视频播放器的播放速度，突破原始网站的限制。

29. pagenote

网页高亮标注

30. 腾讯翻译

chrome 高级搜索
