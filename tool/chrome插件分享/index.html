Chrome插件分享

Chrome 插件不可错过

要发挥 chrome 潜力，安装各种插件是必不可少的。本文整理和分享一下日常使用的插件。

1. 扩展管理器

可以快速的管理扩展，开启和关闭。能够配置情景模式，一键切换扩展组的启用和关闭。另外插件提供了比较实用的工具，可以在特定的网站开启插件。能够节省内存，在不需要的时候，自动开启和关闭插件。比如，github 的 Octree。在 github 网站打开的时候，才会打开插件。

2. infinity pro

新标签页。非常好用。新标签页如同变成小手机。

3. toby mini

这个是一个 tab 存储的插件，能够在打开很多标签的时候，save session，保存所有当下的标签，方便下次打开。mini 版不会占用新标签页。可以和 infinity 配合使用

4. Popup my Bookmarks

管理书签。这样可以将 chrome 浏览器的浏览区域扩大。视觉上也很舒服。

5. Bookmarks clean up

书签清理工具。移除重复的，空文件夹，失效的

6. Tree Style Tab Manager

tab 列表。能够呈现树形结构。方便了解自己的浏览跳转路径。

7. AdGuard

广告拦截插件。内存占用量是 adblock 的一半左右。功能上基本一样。所有可以赶快干掉 adblock 已经 adblock plus

8. LastPass

一款密码管理插件。主动询问保存域名下的密码，并且在登录的时候能够自动帮助填充。拥有账号体系，可以非常方便的在不同的账号下使用。

9. 一键切换

使用搜索引擎的时候，使用快捷键的形式，快速的切换另外一种搜索引擎来搜索现在的关键字。

10. Surfingkeys

和 vimium 类似的强大工具。如果习惯使用 vimium 的，强烈推荐使用。能够有 v 模式。和更强大的 bookmark，history 搜索翻页的能力。还有支持自定义快捷键和一些自定义的脚本。个人自定义了一些东西，可以参考一下

11. 有道词典划词扩展 2019

划词翻译。

12. 腾讯翻译

虽然机器翻译远远比不过，chatgpt，但是，提供的中英文对照还是很舒服的。

13. save-data-header

能够在每个 http 头上添加 accept-coding,告诉服务器，可以返回 br 格式的压缩报文。可以显著的减少支持 br 格式压缩的网站的数据报文。加速打开。

14. Switchy omega

代理切换。

15. Octotree

github 使用的查看仓库源码文件的目录结构。

16. GitCode Tree

码云提供的类似 Ocotree。查看仓库源码文件的目录结构

17. tampermonkey

油猴插件。能够注入 js 脚本。拥有脚本库。有大量编写好的插件，也可以自己写脚本注入。常用的脚本入各大 vip 视频网站的破解。

18. Stylus

提供了对网页css的修改能力。也有丰富的社区社区库。能够改变某些网站的 css 布局和效果。我经常使用 google center search result。使得搜索结果居中。很舒服。

19. quick javascript switcher

有些网站不支持复制。直接禁用 js。大多数就立马投降了。

20. xpath helper

写 xpath 的时候，可以用他来辅助验证自己的 xpath 效果。

21. easy auto refresh

自动刷新页面。

22. editthiscookie

这个你看着用。

23. separate window

基本上算是个摸鱼的神器。能够将网页里面的视频扣出来播放。一一个新的小窗。

24. 购物党

显示一些商品的历史价格，在京东，天猫等购物的时候不至于被坑。

25. 下载管理

chrome 自带的下载管理，设计及其影响全局视觉。通过这个插件可以通过点击插件图标方便查看下载状态。也不会出现原有的下载底栏影响视觉。注意，下关管理的通知使用了 windows 的通知。系统可能因为专注模式，或者通知关闭导致没有通知。

26. astrolabe

可以用快捷键 ctrl+m 预览所有 tab 的缩略图。在打开很多 tab 的时候，可以方便的找到需要的 tab。（可以通过 chrome 自带的扩展的快捷键设置来修改触发的快捷键。本人改成了 ctrl+b.有利于单手操作）

27. darkreader

可以调节网页的亮度，和对比度，找到自己合适的效果。颜色的话，只有黑白，但是亮度和对比度，调节好了之后，基本上颜色也没有啥要求了。因为已经很舒服 了。如果你有绿色背景的需求，这里可以不用担心

28. simpRead 简悦

正文提取算法。提取正文。排除其他信息干扰。阅读更加舒服。

29. Tab Activate

使用新标签页打开所有链接。实际上如果，熟练使用 ctrl 按键。来控制使用新标签打开新标签。方法更加的灵活适用。

30. Video Speed Controller

加速 HTML5 audio/video 视频播放器的播放速度，突破原始网站的限制。

31. pagenote

网页高亮标注

32. pocket

33. rsshub radar

34. 身份验证器
