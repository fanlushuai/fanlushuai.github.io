输入法的问题。

1.经常性快熟敲击键盘的时候，总是失误切换到了微软的难用的垃圾输入法。 原因，系统存在微软输入法，和搜索输入法，并且都存在中英文。

解决，将操作系统级别的输入法切换快捷键禁用掉。使其永不切换，只使用搜狗输入法。之后只需要中英文切换就好。再也不用担心杂乱。

细节：不可将微软输入法卸载掉，这样也能达到效果，但是会出现一些问题，当使用windows的office的软件的时候，甚至还有一些其他潜在问题不可知。 2. 广告，和悬浮的问题。造成桌面很不清爽 设置输入法，去掉所有广告位置，并且将输入法切入状态栏

磁盘问题

  1. 文件单纯存盘，和系统盘分离开 保证，系统出问题，随意格盘，不存在文件丢失。
  2. 下载目录 将所有，具备下载能力的工具，下载目录防止到同一个问题。 chrome，迅雷，百度云,网易云
  3. 工程文件目录 防止自己的产出。
  4. 系统盘符容量分到100G 实践证明，系统盘会保存系统之外的一些用户数据。并且一起缓存文件也在系统盘。如果分配过低，比如60G。将来系统升级，等等缓存文件一叠加，很容易出现内存不够。内存不够的情况下，因为缓存容量的不足。会影响一些软件的性能。

五、windows用户面临的问题

windows一直是我的最爱，我是windows忠实粉丝。尤其是看到这些年windows的变化。 我从windows，xp-vsta-win7-win8-win8,1-win10 一路跟进体验了下来

  1. 软件下载的问题。 没有专门的下载通道，存在很多垃圾下载站点，绑定很多垃圾软件。 建议，官网下载，
  2. 软件的设置问题 大量的广告和垃圾内容。很多都是用户没哟进行继续设置。 建议。用户安装完软件，进行各种设置，调整。
  3. 简洁问题 用户不同得打理自己的系统环境。也说明了，windows系统环境使用的一些高度定制。以及没有特殊的真正的提供给用户舒适的环境。导致用户滥用各种东西。一团糟。
  4. 操作问题 操作问题，其实不是很明显。普通用户是没有问题的。 但是从系统设计的角度来说。统一的全局快捷键设置，统一的手势设置。等等。都没有统一的标准。导致各种软件都是按照自己的想法来。用户自己大多数属于鼠标党。 没啥建议的，如果你注意到这个问题。我想你一定有自己的使用费方法。 我主要是习惯使用vim键位。 chrome的vimsurfing设置，还有虚拟准面的切换快捷键。能给你提供非常强的操作体验。 另外就是vim的键位。
