劝退一个迷茫的人

社交冲突

  1. 生存和生活。人类面临两个问题，第一是生存，第二个是生活。人类文明发展至今，尤其是在中国社会。男权偏多社会。会给男生带来很多压力，但是也看原生家庭。原生家庭的状态，会直接造就，这个家庭中的男性，或者女性。是偏生存性的，还是偏生活性的。最后，会形成两种思想的第一基准，一种是爱人，一种是爱钱（钱是生存）。

  2. 感性和理性。爱人的人，享受过人情的暖，更愿意希望得到感性的力量。理性的人，吃过生存的亏，或者看过生存的不容易，更愿意去挣脱生存问题。两种人生活重心不一样。并且，第一情绪不一样。所以，你会看到，不同的人，在事情上面有不同的态度和说话方式。

  3. 立场和态度。任何事情都有不同的角度，但是基本的角度，一定是有感性和理性的存在。感性一般不是做事的，而是生存一种理想环境，理性一般是冷酷的，但是可以推动事情的进展。每个人情绪的基础都不一样。这种立场从根源上不一样，导致了每个人都有不同的态度。

  4. 除了，大家的最基本的不一样。还有在社会中习得的方式方法的不一样。有时候可以看做是素质问题，行为，也可以看作是一种知识。有的人，积累的够多，就会有优雅的文笔，有的人就会很生硬。但是每个人都有表达的权利。社交，其实是自己表达的问题，也是在看别人表达的问题。其实更多的不仅仅是观点，而是表达方式。实话讲，每个人的修炼都不近相同，表达方式的不一样是一种必然。面对这个无法解决的问题，不应该有苦恼。不应该怨恨自己的不行，也不应该怨恨别人的不行。唯一的就是理解。如果不理解，就只会受到这些的干扰。其实，你想要的太多，想要一个好的环境。这点就违背的客观事实。理解然后顺应。

上面，是我对社交问题的理解。冲突的本源。发展的分支的不一样，以及不同分支对抗的时候的态度。

那么你是一个什么的底层的人，以及你习得了哪些方式？职场，是怎么回事？自己问问自己。

自我冲突。自我价值和人性的弱点。

社交问题是一种外在牵绊，在意识到之后，就可以融入。你会发现，自己还是不开心。因为无法展现自己的价值。那么自己的独特性，是展现给这个世界最好的东西，也是自己的自我价值。

可能部分人都有这种信仰问题，有的想要过好平凡的生活，有的心怀高远的志向。但是这些都是个人最重要的人生观念。

在平静的日子里面，可能都没啥问题。当问题出现之后，人性的弱点就会出现。

  1. 层层构建，并且难以颠覆。 包括社交问题。社交问题的本质，是从自我角度，当然也可以说是，从无意识的自我角度，产生的对自己和社会的态度。因为，很多是从小培养的，是大脑里面注入的第一个概念。自己可能都不知道。之后的一切意识发现，很有可能都会基于这种底层思想，一层层构建。但是万变很难离其宗。所以，有时候，和别人争论的时候，或者朋友抒发自己的感慨的时候，总是可以感觉到，他说一万句不一样的话，你都感觉他只有一个目的。

  2. 压力带来的问题 人的大脑，好比计算服务，有一定的处理能力。当问题过量的时候，可能会宕机。压力的解决办法有很多。有的人学会了逃避，有点人学会了解决。但是，解决一般靠理性，逃避一般来自感性。可以看到底层思想还是在影响着人的后续情绪问题。感性的人，容易出现拖延，逃避，顺带就会产生焦虑，甚至抑郁。对于事务的解决，几乎是啥也没做，一门心思在让自己感觉上好一点。比如，会找朋友排解，等等，这些只是相当于散热的东西。在帮助cpu散热，但是问题还是高频计算未得到解决。大脑依然受不了。然后可以看到，大多数治标不治本。 但是，这里面有没有正确的逻辑？有的。对于感性的人，散热是必须的。但是感性的人，热量会因为问题的到来，产生更多的热量。散不出去，温度降不下来。疑问感性的人十分希望得到外部力量。但是外部力量在大问题面前，不行！风冷的能力远远不如水冷。所以，构建自己的水冷体系变成问题的解决起点。最后，完全冷却下来，没有了感性的困扰，理性自然而然就会出现，帮助你解决问题。

  3. 水冷体系的构建。 想要在高压下不易奔溃。要从多个阶段解决。首先是压力的产生源头。比如，生活习惯混乱，没有正常的作息，和简单健康的生活。然后精力不好。这是个人主机的性能差的第一个原因。容易产生高温的原因。 第二个，就是冷却体系。 每个人都会高温，但是高温冷却，每个人都不一样。高温冷却，需要构建舒服的环境。心理上一定是乐观的轻松的。所以冷却系统，一般是从小无意识养成的乐观，这种父母的功劳。以及个人敏感发现，培养出来的自我暗示。以及自己总结的一些小方法。这些如果足够多，并且足够好。冷却系统就会越强大。当然，这是水冷。自我的冷却系统。其实，还有风冷，最强悍的冷却系统，以自我冷却为主，以外部冷却为辅。 总而言之。冷却有多种方式，是否具备？需要反思和培养。这是下一步解决问题的前提问题。

  4. 解决问题。高效问题 解决问题，实际上在能够持续性的解决问题之后，就已经可以看做不是问题的问题。只是时间问题。 突出一个任务管理的问题。任务的确定，任务的拆解，时间的合理评估，完美配合。以及任务的修正。使得生活进入高效的状态。普通人和天才最大的差别，就在于，在相同的时间能不能走的更快。除了物理的限制，更多的是发挥自己的潜能。

  5. 人生的大问题 人想要怎么过一生，想找怎么样的伴侣，想要成就什么事业，等等。很多价值观念，都会在深入社会，深入自己的过程中，慢慢养成。其实观念一旦养成，很多东西，都是时间问题。

人生其实是自我观念的养成和观念的实现。

目前，形势的要求，是更快的解决问题。不是去深化问题。反而是简化问题。
