使用介绍

函数说明：

  • bootXunFeiFloatWin 启动讯飞悬浮窗
  • switchXunFeiFloatWin 打开或者关闭悬浮窗
  • toSimpleChinese 在启动讯飞悬浮窗的前提下，切换到普通话
  • toSuiShengYi 在启动讯飞悬浮窗的前提下，切换到随声译

使用例子

; 悬浮窗路径
global config_xunfei_location := '"C:\ProgramData\Microsoft\Windows\Start Menu\Programs\讯飞输入法\语音悬浮窗.lnk"'

; 按下1，就会启动悬浮窗，并且选择普通话。如果已经启动了，就直接选择普通话
1:: {
    bootXunFeiFloatWin
    toSimpleChinese
}

2:: {
    bootXunFeiFloatWin
    toSuiShengYi
}

; 开关悬浮窗
3:: switchXunFeiFloatWin

配置要求

  1. 代码里面的变量值：（配置为悬浮窗的路径）

    global config_xunfei_location

  2. 如果，在本机不可用，就需要配置这几个图片，替换为自己本电脑的：

    • xunfei_change_blue.png
    • xunfei_change_gray.png
    • xunfei_putonghua.png
    • xunfei_suishengyi.png

  3. 自己设置键位绑定(上面的，1，2，3 只是个例子)

  4. 以管理员权限，运行脚本
