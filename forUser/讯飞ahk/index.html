使用

配置

  1. 代码里面的变量值：（配置为悬浮窗的路径）

    global config_xunfei_location

    global config_xunfei_key_startVoice

  2. 如果，在本机不可用，就需要配置这几个图片，替换为自己本电脑的：

    • xunfei_change_blue.png
    • xunfei_change_gray.png
    • xunfei_putonghua.png
    • xunfei_suishengyi.png
    • xunfei_suishengyi_cn2en.png
    • xunfei_suishengyi_cn2yue.png
    • xunfei_suishengyi_more.png

  3. 自己设置键位绑定(上面的，1，2，3 只是个例子)

  4. 以管理员权限，运行脚本

例子


; 悬浮窗路径
global config_xunfei_location := '"C:\ProgramData\Microsoft\Windows\Start Menu\Programs\讯飞输入法\语音悬浮窗.lnk"'

; 自己配置的讯飞，语音开启快捷键。
global config_xunfei_key_startVoice := "{F10}"

1:: {
    bootXunFeiFloatWin
    toSimpleChinese
    startInputVoice
}

2:: {
    bootXunFeiFloatWin
    toSuiShengYi
    startInputVoice
}

3:: switchXunFeiFloatWin

; ch2en cn2yue 。type参数的值。是图片名字中的一部分。也就说，自己可以随便扩展任何的
4:: useSuiShengYiOf "cn2en"

5:: useSuiShengYiOf "cn2yue"

; 如果，浮动窗口，自己给他改变位置了。 发现，脚本点击不到了。
; 那么需要清理一下缓存。 将下面的函数绑定即可。
; 或者，直接重启脚本，就可以达到清理缓存的效果
; 4::clearCache

函数说明：

  • bootXunFeiFloatWin 启动讯飞悬浮窗（如果未启动的话，启动）
  • switchXunFeiFloatWin 打开或者关闭悬浮窗
  • toSimpleChinese 在启动讯飞悬浮窗的前提下，切换到普通话
  • toSuiShengYi 在启动讯飞悬浮窗的前提下，切换到随声译
  • useSuiShengYiOf 切换到随声译的，更加具体的语种。
  • clearCache 清理缓存。为了增加性能，脚本只在第一次进行图片搜索，后面就会记录位置。在再次触发的时候，直接点击位置，增加了速度，用户体验更好。但是，缺点，就是浮动窗口移动的话，需要清理一下缓存。或者重启脚本。（之所以，设计缓存，就是因为，这个浮动窗口，看上去，并不会经常移动）
