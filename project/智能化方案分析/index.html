智能化方案分析

一、 智能化认知态度-市场和技术问题：

  1. 现存的电器控制方式交错：插电独立控制且默认关闭状态，插电独立控制默认开启状态，后者可加入智能开关控制。比如，单一控制的灯具。复杂的只能购置相关智能的设备。
  2. 深度控制不易完成，比如，冰箱，有很多控制模式，智能化可能就不能完成。（其实也没必要，因为冰箱大多数都是常开的）传统电器的智能化鸡肋。
  3. 市场上智能化方案，相互独立，未必能够打通。比如，小米，天猫，京东，华为，苹果等，相互之间有一定的隔离性（竞争关系吧），有一些 homesistant 方案，不过覆盖不全面。所以，在采购设备的时候需要注意。一步小心，就会增加整个系统的复杂度，以及带来使用上的割裂感，比如，一个屋子，同时出现，各种智能语音入口，小爱，小度，天猫精灵。等。
  4. 由于市场技术原因，有的智能化比较稳定，有的不是很稳定。作为用户，我们只能妥协。
  5. 灯具市场不易打通。越简单的灯具，越容易打通，越复杂的只能直接上智能化完成的。

所有的智能，都是以语音，手机 app，设备联动以及云端自动化为用户操作入口的。

所以，在选用方案的时候，先要有一个正确认知态度。

二、 方案的选择-依赖市场

  • 技术流：华为智能比较费钱，技术上需要从排线开始，并且设备都需要采购相关的。垄断太厉害。虽然技术一流。很多限制感。
  • 改造流：小米的，比较亲民。各种配件都有，可以相对简单和轻松的实现。但是，技术上稳定性，存在一定问题。不过从普通用户的角度，问题不大。
  • 电商流：天猫精灵，覆盖面比较广，主要是电商平台销售的合作电器，多数直接就内置了天猫精灵控制单元。
  • 京东的也是类似，还有一些其他厂商接入到，小米，天猫，京东的。都属于合作关系。但是智能化做的都一般吧。只能实现基本的。

个人觉得，小米，简单。如果重新装修，可以考虑看看华为，不过价格和复杂度有点高。

三、 目标功能

以有效，实用，为主。虽然大部分都可以实现智能化，但是真的不是所有东西实现智能化之后都会带来便利。有的挺鸡肋。

比如实用的：

  1. 常用电器控制：热水器部分，热水器，风扇，电饭煲，烧水壶。

  2. 光源：灯具，窗帘

  3. 影音：电视，音箱。

  4. 安防：门锁，摄像头。

  5. 懒人专供：扫地机器人。

四、小米方案基本采购：

具体的看实际情况，比如，插排，已存电器控制方式。 大部分如下了。

基础建设配置： 路由器，红外蓝牙 mesh 语音音箱，

开关式设备智能控制： 智能插座若干，大功率智能插座，或者墙体内置智能开关。

智能门锁： 开关门检测，或者购买智能门锁，接入。

观影体验： 电视盒子满足某品牌。可以实现语音控制。

窗帘体验： 直接买智能的窗帘。

室内检测： 比如，温度，湿度，传感器。人体红外传感器。

咋项： 比如，智能加湿器，智能电饭煲，等吧

在基本设备达到一定数量的时候，可以自定义一些联动控制。出现模式的概念。比如，出入门，睡觉起床。

五、费用和时间

主要就是采购时间，每一个设备可能需要自己具体再选选。 安装和调试，可能需要一两天。

设备费用。设备估计需要个几千吧。

如果懂技术，自己直接就搞了，不是特别复杂。
